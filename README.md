This application simulates a basic hotel reservation system and focuses on applying programming fundamentals and system structure.

- Hotel room management
- Basic reservation logic
- Data handling for users and rooms
- Academic-level implementation

## Technologies

- Object-Oriented Programming (OOP)

This project was created for learning purposes to practice:

- Object-oriented design
- Basic business logic
